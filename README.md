# M5lib
Librairie standard pour la flotte de M5Stack gérant les automatisations du FabLab SU

Cette librairie utilise et nécessite pour fonctionner les librairies standards suivantes :
- M5Unified
- M5GFX
- Wifi (de Arduino)
- Wire (de Arduino)
- HTTPCLient
- ArduinoJson
- MFRC522_I2C
- UNIT_UHF_RFID
